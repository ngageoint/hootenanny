[[db-delete]]
== db-delete

=== Description

The +db-delete+ command deletes a map from the Hootenanny Web Services database.

* +mapUrl+  - The URL pointing to the database map.

=== Usage

--------------------------------------
db-delete (mapUrl)
--------------------------------------
