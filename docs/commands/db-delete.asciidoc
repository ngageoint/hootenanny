[[db-delete]]
== db-delete

=== Description

The `db-delete` command deletes maps from the Hootenanny Web Services database.

* `mapDatabaseLayerUrl(s)` - One or more URLs pointing to database maps.

=== Usage

--------------------------------------
hoot db-delete (mapDatabaseLayerUrl1) [mapDatabaseLayerUrl2...}
--------------------------------------
